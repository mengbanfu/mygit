#自用的git指令集

###拉取当前文件内容和推送相关

```bash
git add           #添加相关文件
git commit        #提交文件
git push -u origin master			#推送到Github
```

###拉取远程代码

```bash
git pull
```

###查看全局配置

```bash
git config --global --list
```

###查看当前仓库的远程地址

```bash
git remote -v
```

###查看当前分支

```bash
git branch
```

###查看仓库状态

```bash
git status
```

###连接远程仓库

```bash
git remote add origin 你的仓库地址
```
