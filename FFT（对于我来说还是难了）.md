### FFT（对于我来说还是难了）

#### 1 . 首先是关于欧拉函数（欧拉还是太权威了）

定义 *i*2=−1，复数是形如 *a*+*bi*（*a*,*b*∈*R*）的数，其中 *a* 是实部（*R**e*(*a*+*bi*)=*a*），*b* 是虚部（*I**m*(*a*+*bi*)=*b*）。

运算：

- (*a*+*bi*)±(*c*+*d**i*)=(*a*±*c*)+(*b*±*d*)*i*。
- (*a*+*bi*)(*c*+*d**i*)=(*a**c*−*b**d*)+(*a**d*+*b**c*)*i*。
- *c*+*d**i**a*+*bi*=*c*2−*d*2(*a**c*−*b**d*)+(*a**d*+*b**c*)*i*。

一个复数对应一个复平面上的点，所以也可以用幅角 *θ* 和模长 *r* 确定一个复数。

欧拉公式 *e**i**θ*=cos*θ*+*i*sin*θ*。所以也可以用 *r**e**i**θ* 表示一个复数。

此时复数加减可以看作向量加减，复数乘法可以看作模长相乘、幅角相加。

当 *r*=1 构成复平面上单位圆，*ω**n*=cos(*n*2*π*)+*i*sin(*n*2*π*) 是单位根，表示单位圆上第一个 *n* 等分点。

单位根性质：

- *ω*2*n*2*k*=*ω**n**k*。
- *ω*2*n**k*=−*ω*2*n**n*+*k*。



#### 2.之后对要求的多项式进行奇偶分解

把多项式奇偶分组，则 *A*(*x*)=*A*0(*x*)+*x**A*1(*x*)，根据单位根的性质，对 0≤*k*<*n*：

- *A*(*ω*2*n**k*)=*A*0(*ω**n**k*)+*ω*2*n**k**A*1(*ω**n**k*)。
- *A*(*ω*2*n**k*+*n*)=*A*0(*ω**n**k*)−*ω*2*n**k**A*1(*ω**n**k*)。

可以分治进行，时间复杂度 *O*(*n*log*n*)。

##### 第一步：逆置预分组

```c++
for(int i = 0, j = 0; i < n; i ++ )
{
    if(i < j) swap(a[i], a[j]);
    for(int k = 1 << (m - 1); (j ^= k) < k; k >>= 1) ;
}
```

##### 第二步：计算旋转因子（最简单易懂的一步）

```c++
static Complex w[N];
w[0] = {1, 0};
for(int i = 1; i < n; i <<= 1)
{
    Complex wi = {cos(PI / i), sin(PI / i) * ty};
    for(int j = i - 2; j >= 0; j -= 2)
    {
        w[j] = w[j >> 1];
        w[j + 1] = w[j] * wi;
    }
```

##### 第三步：计算蝴蝶操作

公式：

AB=X+Y⋅W=X−Y⋅W

- `t1 = X`
- `t2 = Y * W`
- 上半：`X+Y*W`
- 下半：`X-Y*W`

```c++
for(int j = 0; j < n; j += i << 1)
for(int k = j; k < j + i; k ++ )
{
    Complex t1 = a[k], t2 = a[k + i] * w[k - j];
    a[k] = t1 + t2, a[k + i] = t1 - t2;
}
```

##### 第四步：合并

```c++
inline void FFT(Complex a[], int n, int m, int ty)
{
    for(int i = 0, j = 0; i < n; i ++ )
    {
        if(i < j) swap(a[i], a[j]);
        for(int k = 1 << (m - 1); (j ^= k) < k; k >>= 1) ;
    }
    static Complex w[N];
    w[0] = {1, 0};
    for(int i = 1; i < n; i <<= 1)
    {
        Complex wi = {cos(PI / i), sin(PI / i) * ty};
        for(int j = i - 2; j >= 0; j -= 2)
        {
            w[j] = w[j >> 1];
            w[j + 1] = w[j] * wi;
        }
        for(int j = 0; j < n; j += i << 1)
            for(int k = j; k < j + i; k ++ )
            {
                Complex t1 = a[k], t2 = a[k + i] * w[k - j];
                a[k] = t1 + t2, a[k + i] = t1 - t2;
            }
    }
    if(ty == -1)
    {
        double t = 1.0 / n;
        for(int i = 0; i < n; i ++ ) a[i].a *= t;
    }
}

```
